Rule Syntax

  • For a CSS file, rules are written for how the targeted elements should be styled on a webpage.
  • Syntax for these rules looks like this:
  • Every rule begins with a selector then followed by curly brackets { }.
  • Inside of thise curly brackets, declarations are made of property: value pairs seperated by a colon.
  • Each line will end with a semicolon ; .
  • CSS files have as many rules as desired. Even most webpages commonly used could have several hundreds of CSS rules for it.
